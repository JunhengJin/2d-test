1. What is Smooth Pixel?

Smooth Pixel is a anti-aliasing shader for pixel art graphics. It reduces visual errors caused by mismatches between the pixel grid on the screen and the pixel grid on a texture. It works best when the ratio between the screen pixels and the texture pixels is greater than 2:1.

2. How to use Smooth Pixel:

Smooth Pixel consists of a single, UI-compatible shader. A Smooth Pixel material created from the shader is provided and usage is shown in the sample scene.

There is only one detail you absolutely need to take care of to make the shader work: the Filter Mode for any texture you plan to use with the Smooth Pixel shader needs to be set to Bilinear in the Import settings. The shader uses bilinear sampling along the edges of the texture pixels to achieve its anti-aliasing.
